Description
-----------
The Cloud-Viewer is an on-developing command-line tool for visualising .pcd format point cloud data

Compiling
---------
### Prerequisites
    pcl-1.9.1

```
git clone git@github.com:CuiRuikai/cloud-viewer.git
cd cloud-viewer
mkdir build
cd build
cmake ..
make
make install
```
